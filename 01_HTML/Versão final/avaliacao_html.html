A tag h1 Cria um título de maior relevância dentro do site
A tag p cria um parágrafo
A tag a cria um link com algum elemento que pode ser externo ou interno.
A tag form cria um formulário e seus atributos são action (para definir a url que receberá os dados) e method.
A tag table cria tabelas, com o uso de outras tags filhas, como tr, th, td.

As tag meta ficam na parte head do código, ou seja, não tem função visual. Elas auxiliam os mecanismos de busca a lerem o site.

Os links relativos linkam duas páginas que existem no mesmo diretório, enquanto o link absoluto linka páginas que podem ou não estar no mesmo diretório.

Uma âncora é um tipo de link que faz referência a algum elemento dentro da própria página. Para sua criação, é importante que o que se deseja ser linkado possua um id que o idenfique e na criação da âncora, esse id deve ser referenciado.

O input tipo radio permite ao usuário selecionar apenas uma das opções de mesmo nome.

O valor que é passado ao backend é o value, neste caso, "veiculo".

A tag utilizada para a criação de coluna é a td; criação de linha é tr; criação de cabeçalho é th

header: cabeçalho de site ou seções; geralmente contém um título, um conteudo introdutório, slogan e logotipo.
nav: menu no site, agrupa links externos e links de âncora.
main: especifica o conteudo principal do documento e deve ser exclusivo para o documento, não podendo ter conteudo repetidos e também deve ser um elemento único do documento.
section: representa uma fatia do layout, agrupando diversos elementos que tenham relação entre si.
article: define um bloco de informações; é mais específico que a section.
aside: agrupa informações relacionadas ao conteudo principal, progagandas, blocos de navegação, etc, na parte lateral da página;
footer: define um rodapé de uma section ou do documento, podendo, então, haver mais de um por página.
